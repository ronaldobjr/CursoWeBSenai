Demonstração de CSS externo

Note que nesse caso não existe nenhum código CSS na página HTML, pois o arquivo que contém as formatações é externo. Para isso você deve criar um arquivo .CSS em uma pasta "Style" e indicá-lo através de um link dentro da HEAD.

Neste caso todos os parágrafos irão herdar as características definidas pelo arquivo .css, note que qualquer arquivo HTML que você criar e indicar o arquivo css externo será formado pelo padrão definido, isso é muito útil para projetos com mais de um arquivo HTML.
